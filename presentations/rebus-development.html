Moving Forward

We now have this shiny new thing, so how are we going to keep it shiny?

  • Test Driven Development
  • Continious Integration
  • Short Release Cycle
  • Patch Tuesdays - For fast moving security

Customer First

We want to put the customer in the driving seat!

    • Bug reports
  • Continual involvement
